# coc-nvim

安装高版本nodejs，apt中都是低版本

```
curl -sL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
```

PlugInstall 后，执行


# nerdtree

大写I，显示隐藏文件
r，刷新文件目录

u，上一级目录
大写c，设置工作目录


# 自定义快捷键

## 切换光标
句首，句尾 $ , 0
单词间切换 w,b
窗口 ctrl+hjkl

## 新增窗口
vsp sp

## 切换页面
ctrl+ui

# 折叠代码块

zf，折叠
