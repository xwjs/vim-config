# coc-nvim

安装高版本nodejs，apt中都是低版本

```
curl -sL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
```

PlugInstall 后，执行


sudo apt-get install exuberant-ctags

# nerdtree

大写I，显示隐藏文件
r，刷新文件目录

u，上一级目录
大写c，设置工作目录


# 自定义快捷键
shift+hjkl，打开终端
ctrl+hjkl，打开窗口
ctrl+方向，设置光标

自动保存文件

# 折叠代码块

zf，折叠
zo，展开
